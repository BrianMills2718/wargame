# Historical Planning Artifacts

These documents were generated by Gemini based on the initial ChatGPT ontology conversation. They describe an earlier architecture that has been significantly revised.

**Current architecture is defined in:** `claude_code_planning/PLAN.md` and ADRs.

**Key divergences from these docs:**
- The system now has a mechanical causal engine (not just LLM judgment)
- Base rate anchoring constrains GM probability output
- Additive layers (ADR-002) replace the "Thin Spine only" approach
- Dynamic agent population (ADR-003) replaces fixed sub-agent roles
- Domain models (scenario-level) are separate from character models (per-actor)
- Scenario spec format is YAML with causal_edges, variable_dynamics, etc.

These files are preserved for historical reference only.
